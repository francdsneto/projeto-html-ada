Conteúdo principal

Tag <header> que representa o cabeçalho de uma página

Tag <nav> que representa a barra de navegação da página

Tag <main> que representa o conteúdo principal da página

Tag <section> que representa uma sessão dentro da página

Tag <article> que representa uma artigo dentro da página, geralmente usado para blocos que são repetíveis, aceita tags header, main e footer

Tag <figure> que é utilizada para envolver uma imagem. Semanticamente faz mais sentido envolver a imagem com um figure do que com um div. E ela possuo uma tag chamda figcaption, que pode ser utilizada, por exemplo, para descrever a imagem.

Tag <aside> que representa uma barra lateral da página - Pode ser colocado dentro ou fora do main

Tag <footer> que representa o rodapé da página

Em questão de estrutura da página HTML não muda nada, porém as tags possuem mais semântica, faz mais sentido, e auxilia nos mecanismos de busca.
